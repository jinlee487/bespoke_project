# 역할 분담

- `최상혁` : 소셜 로그인, 공지 게시판, 고객 치수 관리 게시판, 메인 페이지, 스프링 부트 설계 
- `이필성` : 회원 관리 페이지, 타일즈, AWS 서버 배포, 스프링 부트 설계
- `김시윤` : 회원 가입, 이메일 인증, 스프링 서큐리티 구현
- `이진우` : 스크롤 버튼, 로그인, 비밀번호 찾기, 쇼케이스 페이지, 쇼케이스 관리 게시판
